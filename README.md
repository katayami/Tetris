# Tetris

Браузерный тетрис с поддержкой мобильных устройств.

## Играть

**https://katayami.github.io/Tetris/**

## Управление

### Клавиатура
| Клавиша | Действие |
|---------|----------|
| ← / A | Влево |
| → / D | Вправо |
| ↓ / S | Вниз |
| ↑ / W | Вращение |
| Space | Сброс вниз |
| Escape | Пауза |

### Мобильный
На экране появляются кнопки управления.
